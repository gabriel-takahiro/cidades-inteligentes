Funcionalidades

O programa oferece uma variedade de funcionalidades, que serão descritas a seguir com base nos menus disponíveis no software.

Indicadores

Na seção de indicadores, é possível exibir todos os indicadores cadastrados no banco de dados, cadastrar novos indicadores no banco de dados, bem como editar ou excluir indicadores já existentes.

Mostrar indicadores

Ao selecionar a opção de exibir indicadores, uma nova interface será exibida, contendo uma tabela com informações como o código, nome, meta, método de cálculo e indicador padrão. Nessa interface, serão exibidos todos os indicadores cadastrados no banco de dados local.

Figura 1: Interface que exibe os indicadores

Cadastrar indicadores

Para cadastrar um novo indicador, uma interface será exibida, onde é necessário preencher todas as informações necessárias, que podem ser vistas na seção Banco de Dados. Para evitar confusões ao inserir o método de cálculo, ele é preenchido por meio de botões disponíveis, como números de 0 a 9, / (divisão), x (multiplicação), + (adição), - (subtração), abre e fecha parênteses (para precedência) e colchetes (para valores inteiros positivos), além de um botão para adicionar indicadores, que abre uma nova interface mostrando os indicadores disponíveis e um campo para digitar o código do indicador escolhido. Há também um botão para apagar o último elemento no método de cálculo, um botão para limpar os campos preenchidos e um botão para concluir o cadastro do indicador.

Figura 2: Interface de cadastro de indicadores

Editar indicadores

Para editar um indicador, é exibida uma interface semelhante à que exibe os indicadores disponíveis, porém com uma coluna adicional para selecionar os indicadores. Para excluir indicadores, basta selecioná-los e clicar em "excluir". Em seguida, uma interface será exibida, com uma tabela contendo todos os indicadores selecionados para a exclusão do banco de dados, além de um botão de confirmação. Para atualizar os indicadores, é necessário selecionar apenas um indicador e clicar no botão "atualizar". Será exibida uma interface semelhante à de cadastro de indicador, mas com os valores do indicador que deseja editar preenchidos. É possível fazer as mudanças desejadas e clicar em "atualizar indicador".

Figura 3: Interface que edita os indicadores

Consultas

Na seção de consultas, é possível realizar uma busca no banco de dados para exibir os resultados previamente calculados dos indicadores.

Consultar indicadores

Para realizar a consulta, é necessário informar o ano, o CEP correspondente ao município e selecionar os indicadores para os quais se deseja obter os resultados. Após preencher esses parâmetros, é possível escolher como deseja que os dados sejam exibidos, seja por meio de uma tabela de resultados ou de um gráfico de barras. Para obter mais informações sobre o funcionamento dos resultados, consulte a seção dos resultados.

Cálculos

Na seção de cálculo, é possível realizar o cálculo dos indicadores de cidades inteligentes.

Calcular indicadores

Para realizar o cálculo dos indicadores de cidades inteligentes é preciso selecionar os indicadores que deseja calcular e escolher se quer realizar o cálculo para um município específico ou para todos os municípios. Caso selecione a opção para um município específico, será necessário informar o CEP do município, o ano para o cálculo dos indicadores e a quantidade de anos para a busca retroativa das variáveis. Se selecionar a opção para todos os municípios, não será necessário informar o CEP. Após inserir todas as informações necessárias, clique em Consultar para realizar o cálculo dos indicadores.

Busca retroativa das variáveis: A busca retroativa das variáveis é um recurso que permite o uso de valores antigos mesmo após o período de atualização ter expirado. Por exemplo, se uma variável teve seu valor disponibilizado em 2020 com período de atualização de 10 anos, ela continuará válida até 2030. Contudo, após 2030, ela não será mais considerada pelo programa para o cálculo de um indicador em 2031. Com o recurso de busca retroativa, é possível utilizar o valor de 2020 em 2031, caso o valor de 2030 ainda não esteja disponível. Esse recurso foi incluído devido ao adiamento do censo de 2020, que é realizado a cada 10 anos e foi afetado pela pandemia. O campo de busca retroativa aceita valores numéricos e permite prolongar o período de atualização das variáveis.

Figura 14: Interface que calcula os indicadores

Resultados

Ao finalizar o cálculo dos indicadores, uma nova interface será exibida para escolher o método de exibição dos resultados: tabela ou gráfico de barras.

Figura 15: Interface de escolha do método de exibição dos resultados

Ao selecionar a opção de exibição dos indicadores em forma de tabela, uma nova interface será apresentada com duas tabelas distintas. Na tabela superior, todos os indicadores calculados serão exibidos em uma tabela com informações detalhadas, incluindo o código do indicador, número da ODS, número da meta, nome do indicador, código do município, nome do município, nome da unidade federativa, ano, resultado, valor oficial e indicador padrão. Indicadores calculados somente com valores oficiais serão marcados como "true" e terão sua cor na tabela em verde, enquanto que indicadores marcados como "false" terão a cor vermelha. Indicadores padrão serão marcados com a cor azul, e os demais com a cor amarela.

Na tabela inferior, serão apresentados os indicadores que ainda não foram calculados. Nessa tabela, é possível obter informações relevantes, como o código e nome do indicador, método de cálculo, código e nome da variável, ano, valor da variável e valor oficial. É possível editar as variáveis que estão sem valor ou com o valor oficial "false", para então recalcular o indicador com as novas informações. Para isso, basta inserir o novo valor no campo "valor da variável", selecionar a variável para armazenar o novo valor no banco de dados e, em seguida, selecionar o indicador para recalcular e clicar em "calcular". As linhas que correspondem a cada indicador serão destacadas em amarelo, enquanto as linhas das variáveis usadas no cálculo serão exibidas em branco.

Figura 16: Interface para exibição dos resultados em forma de tabelas

Cada indicador apresenta um botão que, ao ser acionado, abre uma interface que exibe as variáveis utilizadas no cálculo do indicador. Nessa interface, são mostradas informações como o código do indicador, ano, método de cálculo e resultado para o indicador. Para cada variável, são exibidos o código, nome, valor, ano, valor atualizado e valor oficial. Além disso, a interface conta com dois botões: "Excluir resultado" e "Recalcular indicador". O botão "Excluir resultado" remove o valor do indicador no banco de dados e transfere o indicador para a tabela na parte inferior. Já o botão "Recalcular indicador" só funciona para os indicadores que não foram calculados com os valores oficiais (ou seja, aqueles que possuem o valor oficial igual a "false"). Ao clicar em "Recalcular indicador", ele será transferido para a tabela inferior, mas seu valor permanecerá salvo no banco de dados.

Figura 17: Interface que exibe as variáveis utilizadas no cálculo do indicador

Quando a opção de exibição dos indicadores em formato de gráfico de barras é selecionada, uma nova interface é apresentada, contendo um gráfico de barras e uma legenda que apresenta o nome e o código dos indicadores correspondentes. No gráfico, apenas os indicadores que possuem resultado e valores entre 0 e 100 serão mostrados. As barras são coloridas de acordo com o seu valor, onde as barras vermelhas correspondem aos valores de 0 a 30, as barras amarelas correspondem aos valores de 30 a 70, e as barras verdes correspondem aos valores de 70 a 100. Ademais, é possível filtrar o gráfico de acordo com as ODS dos indicadores.

Figura 18: Interface para exibição dos resultados em forma de gráfico de barras.

Recalcular indicadores

A interface de cálculo de indicadores é responsável por fazer as buscas das variáveis dos indicadores nos bancos de dados públicos e, em seguida, realizar o cálculo do indicador. Esse processo é aplicado apenas para os valores das variáveis e dos indicadores que não estão cadastrados no banco de dados. Isso significa que, se uma variável ou um indicador já tiver um valor no banco de dados, a interface não busca um novo valor e não recalcula o indicador.

Por outro lado, a interface que recalcula indicadores realiza uma busca completa nas bases de dados, independentemente de os valores já estarem cadastrados ou não. Ela busca os valores atualizados de todas as variáveis e recalcula todos os indicadores, independentemente de já terem sido calculados anteriormente. Isso garante que todos os valores estejam atualizados e que todos os indicadores estejam corretos.

Em resumo, a principal diferença entre as duas interfaces é que a interface que recalcula indicadores busca os valores atualizados de todas as variáveis e recalcula todos os indicadores, enquanto a interface de cálculo de indicadores realiza o cálculo apenas para valores que não estão cadastrados no banco de dados.

Figura 19: Interface que recalcula os indicadores

Banco de dados

A seção de banco de dados permite a criação, exclusão e importação de tabelas no banco de dados.

Criar tabelas

Ao selecionar a opção "Criar tabelas", todas as tabelas necessárias para o funcionamento do software serão criadas no banco de dados, caso ainda não existam. É possível visualizar as tabelas criadas na seção Banco de Dados.

Excluir tabelas

Ao selecionar a opção "Excluir tabelas", todas as tabelas criadas no banco de dados serão removidas, independentemente de estarem preenchidas ou não. É importante ter cuidado ao utilizar essa opção, pois todas as informações sobre os indicadores de cidades inteligentes disponíveis no banco de dados serão apagadas.

Importar tabelas

A opção de importar tabelas permite a importação de todas as tabelas incluídas no conjunto inicial de dados fornecido pelo software. Esse conjunto de dados contém informações completas sobre as tabelas de municípios e ODS, bem como variáveis e indicadores considerados padrão. Cada tabela é um arquivo diferente no formato .csv com os respectivos nomes TabelaMunicipios.csv, TabelaODS.csv, TabelaVariaveis.csv e TabelaIndicadores.csv. Esses arquivos podem ser baixados na seção Instalação. Durante a importação dessas tabelas, cada campo é separado pelo símbolo de barra vertical "|" para garantir o correto funcionamento do software. Para visualizar os campos de cada tabela importada, basta acessar a seção "Importados" na área de Banco de dados.
