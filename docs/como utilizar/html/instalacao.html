Instalação

  1. Para utilizar este programa, é necessário ter instalado em seu computador as seguintes ferramentas:

    • PostgreSQL: É necessário para a manipulação dos dados durante o uso do software.

    • JDK 17: É necessário para o funcionamento do software.

  2. Depois de instalar as ferramentas mencionadas acima, baixe o executável do programa.

  3. Baixe todas as planilhas disponíveis em: https://github.com/gabriel-takahiro/cidades-inteligentes/tree/main/Dados

    Salve todas as planilhas e o executável em um mesmo diretório no seu computador.
    Para cada planilha, execute o seguinte procedimento:

    1. Clique com o botão direito do mouse no arquivo baixado.
    2. Selecione "Propriedades" no menu que aparece.
    3. Na janela de propriedades, clique na aba "Segurança".
    4. Clique em "Editar" para modificar as permissões.
    5. Adicione o usuário "Usuários" à lista de permissões.
    6. Clique em "OK" para salvar as alterações.
    7. Permita o controle total para o usuário "Usuários".

Aprenda a utilizar o programa acessando a seção Como usar.
